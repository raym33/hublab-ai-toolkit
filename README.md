## üîß Technical Details for AI Integration

### Capsule Structure
Every capsule has this structure:
```typescript
{
  id: 'capsule-id',
  name: 'Capsule Name',
  description: 'What it does',
  category: 'category-name',
  props: [
    {
      name: 'propName',
      type: 'string' | 'number' | 'boolean' | 'array' | 'object',
      required: true | false,
      default: 'default value',
      description: 'What this prop does'
    }
  ],
  dependencies: [], // Required libraries
  code: `React component code`
}
```

### Export Formats

**React Component:**
```javascript
// Standalone React component
import React from 'react';

export function ComponentName({ prop1, prop2 }) {
  return (
    <div className="...">
      {/* Component JSX */}
    </div>
  );
}
```

**Next.js Page:**
```javascript
// Next.js 14 page component
export default function Page() {
  return (
    <main>
      {/* All components combined */}
    </main>
  );
}
```

**HTML/CSS:**
```html
<!-- Static HTML with Tailwind CSS -->
<!DOCTYPE html>
<html>
  <!-- Complete HTML with all components -->
</html>
```
