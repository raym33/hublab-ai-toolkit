## Step 1: Build the UI (10 minutes)

### 1.1 Open Studio V2
Visit: https://hublab.dev/studio-v2

### 1.2 Add Components

Click **"Add Component"** and select these in order:

1. **Dashboard Layout** (from Layouts category)
   - This creates the main container with sidebar

2. **Stat Card** (from Data Display category)
   - Add 4 of these for your statistics
   - Configure each:
     - Card 1: Title "Total Users", value "2,543"
     - Card 2: Title "Active Now", value "127"
     - Card 3: Title "Revenue", value "$12,345"
     - Card 4: Title "Growth", value "+23%"

3. **Line Chart** (from Charts category)
   - Shows user growth over time

4. **Data Table** (from Data Display category)
   - Displays user list with search

### 1.3 Preview
Click **"Preview"** to see your dashboard live

## Step 4: Connect Real Data (10 minutes)

### 4.1 Extract the ZIP
Unzip to your computer

### 4.2 Open in VS Code (or any editor)
```bash
cd dashboard
code .
```

### 4.3 Install Dependencies
```bash
npm install
npm install swr  # For data fetching
```

### 4.4 Connect to Real API

Open `page.tsx` and add at the top:

```typescript
'use client';
import useSWR from 'swr';

const fetcher = (url: string) => fetch(url).then(res => res.json());

export default function DashboardPage() {
  // Fetch real data
  const { data: stats, isLoading } = useSWR('/api/stats', fetcher);
  const { data: users } = useSWR('/api/users', fetcher);

  if (isLoading) return <div>Loading...</div>;

  // ... rest of your component (already generated by HubLab)
}
```

### 4.5 Create Mock API Routes

Create `app/api/stats/route.ts`:
```typescript
export async function GET() {
  return Response.json({
    totalUsers: 2543,
    activeNow: 127,
    revenue: 12345,
    growth: 23
  });
}
```

Create `app/api/users/route.ts`:
```typescript
export async function GET() {
  return Response.json([
    { id: 1, name: 'John Doe', email: 'john@example.com', status: 'active' },
    { id: 2, name: 'Jane Smith', email: 'jane@example.com', status: 'inactive' },
    // ... more users
  ]);
}
```

## ðŸ’¡ Common Questions

### Q: Can I modify the exported code?
**A:** Yes! You own 100% of the code. It's just React/TypeScript with no proprietary dependencies.

### Q: What if I need custom functionality?
**A:** HubLab handles the UI. Add your custom logic in API routes, hooks, or utility functions.

### Q: Is this production-ready?
**A:** Yes! The code includes:
- TypeScript types
- Error handling
- Loading states
- Responsive design
- Accessibility features

### Q: Can I use my own backend?
**A:** Absolutely! Replace the mock API routes with calls to your existing backend.

### Q: How do I update components later?
**A:** Export again from HubLab and copy the updated components. Your custom logic stays intact.
